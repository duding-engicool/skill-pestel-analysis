# PESTEL 六维度关键思考点清单

> 用于采集门中提示用户"从哪些角度想"，不替代用户判断。

## P 政治 Political
- [ ] 政府稳定性 / 政策连续性
- [ ] 产业扶持 or 限制政策
- [ ] 税收 / 财政导向
- [ ] 贸易政策、关税、进出口管制
- [ ] 国际关系 / 地缘风险
- [ ] 政府对特定技术或行业的态度

## E 经济 Economic
- [ ] 宏观经济 / 行业增速
- [ ] 通胀、利率、汇率
- [ ] 劳动力 / 原材料成本
- [ ] 消费力与需求景气
- [ ] 融资环境 / 资本成本
- [ ] 产业链上下游景气

## S 社会 Social
- [ ] 人口结构 / 代际变化
- [ ] 消费观念 / 价值取向
- [ ] 文化 / 地域偏好
- [ ] 公众舆论 / 品牌敏感度
- [ ] 劳动力素质 / 用工观念
- [ ] 生活方式变迁

## T 技术 Technological
- [ ] 关键技术成熟度
- [ ] 研发 / 创新投入方向
- [ ] 数字化 / 自动化 / 智能化
- [ ] 新工艺 / 新材料替代
- [ ] 技术壁垒 / 专利格局
- [ ] 基础设施（网络 / 算力等）

## E 环境 Environmental
- [ ] 环保 / 双碳政策
- [ ] 绿色制造 / ESG 要求
- [ ] 资源（水 / 能 / 料）约束
- [ ] 气候变化对供应链影响
- [ ] 公众环境诉求
- [ ] 循环经济 / 回收要求

## L 法律 Legal
- [ ] 行业准入 / 产品强制标准
- [ ] 劳动法 / 用工合规
- [ ] 知识产权 / 反垄断
- [ ] 数据与隐私合规
- [ ] 进出口 / 海关法规
- [ ] 行业专项立法动向

> 提示：每个维度建议收集 **4-8 条**要素；过多则聚焦关键，过少则提示"可能遗漏重要信号"。
